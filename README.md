## Deploy using Cloud Foundry CLI
---
* cd to a directory you want the application in.
* ```git clone https://hub.jazz.net/git/ajlohr/Microservices_OrdersAPI```
* cd into the application.
* ```cf login```
* ```cf push <name of your application>```
